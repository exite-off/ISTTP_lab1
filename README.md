# Система обліку інвентарю (InventoryMVC)

## Постановка задачі

Розробити веб-застосунок для автоматизованого обліку інвентарних предметів організації.  
Система дозволяє адміністратору вести облік предметів, відповідальних осіб, відділів, кімнат, категорій та постачальників.

### Основні функції

- **Предмети**: додавання, редагування, видалення інвентарних предметів із зазначенням інвентарного номера, назви, дати введення, гарантійного терміну, ціни, статусу, категорії, постачальника, кімнати та відповідальної особи.
- **Відповідальні особи**: управління переліком працівників, закріплення за відділами та предметами. При видаленні особи — перевірка відсутності закріплених предметів.
- **Відділи**: ведення структури відділів організації, їх адреси та кімнат.
- **Журнал змін**: автоматична фіксація передачі відповідальності та інших змін по кожному предмету.

### Обмеження предметної області

- Відповідальна особа **не може бути видалена**, якщо за нею закріплено хоча б один предмет.
- Кожен інвентарний предмет має унікальний інвентарний номер.
- Гарантійний термін та опис є необов'язковими полями.
